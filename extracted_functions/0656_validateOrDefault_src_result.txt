## Function Content
```zig
    pub fn validateOrDefault(
        comptime T: type,
        value: T,
        min: T,
        max: T,
        default: T,
        logger: *log_mod.Logger,
    ) T {
        if (value < min or value > max) {
            logger.warn(
                "Value {any} out of range [{any}, {any}], using default {any}",
                .{ value, min, max, default },
            );
            return default;
        }
        return value;
    }
```

## Function Content
```zig
    pub fn getEndTick(self: *const Tuplet) u32 {
        if (self.notes.len == 0) return self.start_tick;
        const last_note = self.notes[self.notes.len - 1];
        return last_note.start_tick + last_note.duration;
    }
```

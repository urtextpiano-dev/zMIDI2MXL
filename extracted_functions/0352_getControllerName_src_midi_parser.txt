## Function Content
```zig
    pub fn getControllerName(self: ControlChangeEvent) []const u8 {
        return switch (self.controller) {
            7 => "Volume",
            11 => "Expression",
            64 => "Sustain Pedal",
            else => "Controller",
        };
    }
```

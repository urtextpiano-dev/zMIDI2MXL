## Function Content
```zig
    pub fn toString(self: KeySignatureEvent, buffer: []u8) ![]const u8 {
        const key_name = self.getKeyName();
        const mode = if (self.is_minor) "minor" else "major";
        return std.fmt.bufPrint(buffer, "{s} {s}", .{ key_name, mode });
    }
```

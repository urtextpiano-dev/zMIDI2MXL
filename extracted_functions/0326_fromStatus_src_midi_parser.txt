## Function Content
```zig
    pub fn fromStatus(status: u8) MidiEventType {
        return switch (status & 0xF0) {
            0x80 => .note_off,
            0x90 => .note_on,
            0xA0 => .polyphonic_pressure,
            0xB0 => .control_change,
            0xC0 => .program_change,
            0xD0 => .channel_pressure,
            0xE0 => .pitch_bend,
            else => .other,
        };
    }
```

## Function Content
```zig
    pub fn getControllerType(self: ControlChangeEvent) ?ControllerType {
        return switch (self.controller) {
            64 => .sustain_pedal,
            7 => .channel_volume,
            11 => .expression,
            else => null,
        };
    }
```

## Function Content
```zig
    fn writeEscapedText(self: *XmlWriter, text: []const u8) !void {
        for (text) |char| {
            switch (char) {
                '&' => try self.writer.writeAll("&amp;"),
                '<' => try self.writer.writeAll("&lt;"),
                '>' => try self.writer.writeAll("&gt;"),
                else => try self.writer.writeByte(char),
            }
        }
    }
```
